Introduction

The Gemini Manifold Book

Before we can teach a model to communicate with ASCII art, we need to know what the model thinks ASCII art is. Does it read an ASCII cathedral mainly as text, mainly as image, or as something that can bridge both? Gemini's multimodal embedding space gives us a way to test that question directly.

This book is where that measurement work is documented. Each chapter fixes one experiment in public, stores the embeddings, and builds the visual aids needed to think clearly about what the geometry is actually showing instead of hand-waving around it.

The Measurement Problem

One concept can have many semantic bodies.

A single ASCII artifact can be embedded as raw text, as a rasterized image, and as an interleaved multimodal object. A real photograph of the same subject can be embedded too, along with hybrid objects that combine the photo with ASCII-derived forms. Each of those is a separate point in the same vector space. The distances between those points are the observation.

Method

How the current experiment is built

  • Embed the ASCII cathedral as text.
  • Rasterize the same ASCII and embed it as image.
  • Embed the same artifact as image+text.
  • Embed a reference JPG photo of Saint Basil's Cathedral as photo.
  • Embed two bridge objects: photo+text and photo+ascii-image.

Observed Result

In this first case, visual structure bridges to the photo more strongly than raw ASCII text does.

The strongest evidence so far is not rhetorical. It is numerical. In the current six-object manifold, the raw ASCII text object is the farthest point from the photo object, while the image-derived and hybrid objects sit substantially closer.

Farthest pair now text ↔ photo cos 0.3799, l2 1.1137
Photo closest to photo ↔ photo+ascii-image cos 0.8121, l2 0.6130
ASCII bridge image ↔ image+text cos 0.7925, l2 0.6442

What We Can Say

Claims the current evidence supports

  • ASCII does not collapse to one reading. The same artifact occupies several distinct positions.
  • The rasterized ASCII image is much closer to the photo-derived region than raw ASCII text is.
  • Hybrid objects create bridge points instead of trivially collapsing onto one modality.

What We Cannot Yet Say

Questions still open

  • We do not yet know whether this geometry generalizes beyond one cathedral.
  • We do not yet know whether triangle size is a reliable coherence metric across a corpus.
  • We do not yet know whether overlays, programs, or training trajectories will produce cleaner reward signals.

Why A Book

The work is to build instruments before building doctrine.

The temptation is to jump straight to big claims about teaching models to speak in ASCII. This book takes the slower path. First build the corpus. Then store the embeddings. Then measure every relation we can think of. Then create enough visual aids that the space becomes cognitively manageable. If the stronger theory is real, it should survive that discipline.
